# ghostty

> terminal emulator that differentiates itself by being fast, feature-rich, and native

## install

```sh
brew install --cask ghostty
```

## usage

```sh
ghostty +version
ghostty +help
ghostty +list-fonts
ghostty +list-keybinds
ghostty +list-themes
ghostty +list-colors
ghostty +list-actions
ghostty +ssh-cache
ghostty +edit-config
ghostty +show-config
ghostty +validate-config
ghostty +show-face
ghostty +crash-report
ghostty +boo
ghostty +new-window
```
