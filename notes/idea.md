# idea

> python scrip starting up intellij

## install

```sh
# Tools->Create Command-Line Launcher...
```

## option

```sh
-h, --help      # -
-w, --wait      # -
```

## usage

```sh
idea DIR

idea diff LEFT RIGHT        # launch diff tool

idea merge LOCAL REMOTE
```
