# elm

> functional language that compiles to [[javascript]]

## install

```sh
brew install elm
```

## usage

```sh
elm repl    --help
elm init    --help
elm reactor --help
elm make    --help
elm install --help
elm bump    --help
elm diff    --help
elm publish --help
```

## repl

```sh
> 42 / 7.5
5.6 : Float

> pi
3.141592653589793 : Float

> List.reverse [ "Next", "Stop", "Pottersville" ]
["Pottersville","Stop","Next"] : List String
```
