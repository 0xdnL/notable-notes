# rg

> ripgrep - recursively search the current directory for lines matching a pattern

alternative to: [[grep]]

[[uutils]]

## install

```sh
brew install ripgrep
apt-get install ripgrep
```

## usage

```sh
rg -n -w '[A-Z]+_SUSPEND'
```
