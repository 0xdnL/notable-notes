# bw

> cli for bitwarden

## install

```sh
brew install bitwarden-cli
```

## env

```sh
BW_SESSION        # session
BW_CLIENTID       # api client_id
BW_CLIENTSECRET   # api client_secret
```

## usage

```sh
bw login

bw list folders

bw 
```
