# helmfile

> declarative spec for deploying helm charts

## install

> needs [[helm]] and plugins `helm diff`, `helm secrets`, `helm helm-git`, `helm s3` which can be installed via init

```sh
brew install helmfile
```

## usage

```sh
helmfile init

...
```
