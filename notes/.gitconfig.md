# .gitconfig

> config file for [[git]]

## seperate configs per remote

```sh
cat <<EOF > ~/.gitconfig
[core]
  editor = vim

[includeIf "gitdir:~/gitlab.com/"]
  path = ~/.gitconfig-gitlab

[includeIf "gitdir:~/github.com/"]
  path = ~/.gitconfig-github
EOF

cat <<EOF > ~/.gitconfig-github
[user]
  name  = user
  email = user@mail.com

[push]
  default = simple
EOF
```
