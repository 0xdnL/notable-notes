# ln

> make links between files

## usage

```sh
ln -s SOURCE_FILENAME SYMBOLIC_FILENAME    # create a soft link named link1 to a file named file1

ls -l FSOURCE_FILENAME SYMBOLIC_FILENAME   # verify new soft link 
```
