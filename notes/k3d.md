# k3d

> lightweight wrapper to run [[k3s]] in [[docker]] and minimal Kubernetes distribution for rancher desktop

## install

```sh
curl -s https://raw.githubusercontent.com/k3d-io/k3d/main/install.sh | bash
```

## usage

```sh
k3d cluster create CLUSTER_NAME
```
