# dmidecode

> DMI table decoder  - tool for dumping a computer's DMI (some say SMBIOS ) table contents in a human-readable format.


## usage

```sh
-t, --type TYPE         #
```

```sh
dmidecode -t 1          # 1 = System 

dmidecode --type BIOS
```
