# gpgconf

> gpgconf - Modify .gnupg home directories

## usage

```sh
gpgconf --list-components       # list all components

gpgconf --kill COMPONENT        # kill a component i.e. gpgconf --kill dirmngr

gpgconf --kill all              # kill all components
```
