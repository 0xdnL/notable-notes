# gdb

> gnu debugger

## install

```sh
apt install gdb
brew install gdb    # x86_64 architecture is require !
```

## option

```sh
-q, --quiet, --silent     # won't print warranty etc.
```

## usage

```sh
gdb --version

gdb -q BINARY
```
