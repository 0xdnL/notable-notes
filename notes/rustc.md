# rustc

> rust compiler

## install

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## usage

```sh
rustc --version

rustc FILE.rs               # compile rust

rustc --explain E0382       # explain
```
