# vim tabs

```sh
:tabnew filename		  # load file in new tab

:tabm n
:tabm 2		            # move current tab to position 2

gt                    # pext tab
gT                    # prev tab
[nnn] gt              # numbered tab


:Sexplore
:Sex        # SplitExplore

:Vex        # VerticalExplore
:Tex        # TabExplore    <=== *** this is the one ***
:Ex
```
