# ksh

> korn shell

## option

```sh
-c COMMAND              # command string 
-i                      # interactive mode
-l                      # login shell
-s                      # shell reads commands from STDIN; all non-option arguments are positional parameters
-r                      # restricted mode
```

## usage

```sh
ksh       # start korn shell
```
