# openssl digest

> dgst - Message Digest Calculation

## usage
```sh
openssl md5 FILENAME
openssl sha1 FILENAME

openssl dgst -md5 FILENAME
openssl dgst -sha1 FILENAME
openssl dgst -sha256 FILENAME
```
