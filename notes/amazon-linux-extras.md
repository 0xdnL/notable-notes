# amazon-linux-extras

> manage specific fresh software

## install

`yum install -y amazon-linux-extras`

## usage

```sh
amazon-linux-extras                             # list

amazon-linux-extras enable php7.2               # enable desired topic

amazon-linux-extras install -y postgresql12     # 
```
